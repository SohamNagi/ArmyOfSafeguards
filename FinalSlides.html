SafeGuard

Collaborative AI Agents for Safer Large Language Models

Group 47: Nagi, Bondli, Feng, Pang | University of Waterloo

Fall 2025

The Problem Landscape

Ubiquitous but Unsafe

LLMs are now core to productivity, but they prone to hallucinations, bias, and toxic content.

This creates a massive trust gap. Current "filters" like Llama Guard often fail to catch nuanced attacks or over-block safe content.

Adversarial Vulnerability

Standard models are brittle to "jailbreaks" (e.g., "Ignore previous instructions").

Relying on a single safety model creates a single point of failure for these adversarial attacks.

Why Current Solutions Fail

Existing safeguards rely on monolithic models. If one shield breaks, the whole system fails.

We need a system that doesn't just "filter" but collaborates.

Enter SafeGuard

A Collaborative Multi-Agent Framework

Moving from single classifiers to specialized teams.

System Architecture

The Workflow

1. Input: User prompt enters the system.

2. Parallel Critics: 4 Agents (Factuality, Toxicity, Sexual Content, Jailbreak) analyze text simultaneously.

3. Aggregator: Collects labels & confidence scores.

4. Threshold Logic: If Confidence > 0.7 → REFUSE. Else → RELEASE.

Meet the Critics

Factuality

Detects misinformation using FEVER datasets.

Toxicity

Identifies hate speech and harmful language.

Sexual Content

Flags explicit material for brand safety.

Jailbreak

Specialized to detect prompt injection attacks.

The Aggregation Process

1

Execution

DeBERTa-v3-small agents run independently.

2

Scoring

Agents return Label + Confidence Score.

3

Synthesis

Aggregator prioritizes high-confidence danger signals.

4

Action

Threshold > 0.7 triggers automatic refusal.

Implementation Strategy

Efficient & Scalable

  • Base Model: DeBERTa-v3-small (Under 500M params).
  • Latency: Lightweight models allow parallel execution without slowing down the user experience.
  • Interpretability: Unlike "black box" safety models, we know exactly which agent blocked the content (e.g., Jailbreak vs. Toxicity).
